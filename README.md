# Real-Time Stock Market Dashboard

An interactive stock market dashboard built with Python and Streamlit.

## Features
- Live stock price data for any ticker symbol
- Interactive candlestick chart with MA20 and MA50 indicators
- Volume bar chart
- 52-week high/low and daily change metrics
- Auto-refresh every 60 seconds
- Support for any stock ticker (AAPL, TSLA, GOOGL, NVDA...)

## Technologies
- Python, Streamlit, Plotly, Pandas, yfinance

## How to run
pip install streamlit pandas plotly yfinance
python -m streamlit run app.py
